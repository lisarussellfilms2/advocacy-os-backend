# advocacy-os-backend

Cloud-native AI agent architecture powering ArtsEnvoy.ai.

## Overview

This repository contains the modular backend structure for an AI-native advocacy production system.

The system translates:

Policy Input → Narrative Blueprint → Campaign Assets

Built as a scalable API-first architecture for institutional deployment.

---

## Architecture

- FastAPI application layer
- LLM orchestration wrapper (model-agnostic)
- Structured narrative reasoning modules
- Token optimization layer
- MongoDB persistence
- Containerized for Cloud Run deployment

---

## Core Modules

- Policy Ingestion Layer
- Narrative Blueprint Generator
- Speech & Script Generator
- Campaign Planner
- Asset Packaging Layer

---

## Deployment

Designed for:
- Google Cloud Run
- Docker containerization
- Environment-based model switching (Gemini / Claude-ready)

---

## Status

Production MVP deployed and undergoing optimization for institutional licensing.
